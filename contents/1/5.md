## 答案

### 1 方向变换
```
>>> 51//4
12
>>> 51%4
3
```
**答: 向左转51次之后,**
**小明面朝东方，**
**转过了12圈。**

解析：
由于每左转四次都会回到原来朝向，
所以左转51次和左转3次效果一样，
面朝北方，左转3次后将面朝东方。

### 2 使用循环输出字符串1
参考答案：
```python
for i in range(5):
    part1 = "A"*(5-i)
    part2 = " "*3
    part3 = "A"*(i+1)
    s = part1 + part2 + part3
    print(s)
```

### 6 求和*
参考答案：
```python
s = 0
k = 1
for i in range(100):
    s += k / (i + 1)
    k = -k

print(s)
```
